## 🔧 Local Development
1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Open with live server or serve locally
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```

3. Open `http://localhost:8000` in your browser
